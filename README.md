## 🔧 프로젝트 소개

### 주요 기능
- 익명성을 보장하며 CRUD(Create, Read, Update, Delete) 기능 구현
- 게시물 작성, 수정, 삭제 및 답변 기능 제공
- 좋아요/싫어요와 같은 SNS 기능으로 사용자 반응 기록
- 타 플랫폼과 연동된 링크 공유 기능
